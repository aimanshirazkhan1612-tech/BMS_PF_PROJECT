# Bank Management System

## Project Description
This is a Bank Management System developed in C language using arrays, functions, loops, switch cases, and conditional statements.

## Features
- Admin Login
- Employee Login
- Customer Sign Up
- Customer Login
- Account Verification (Approve/Reject)
- Deposit Money
- Withdraw Money
- Balance Inquiry
- Transaction History
- Complaint Management
- Customer Reapply Request
- Employee Management
